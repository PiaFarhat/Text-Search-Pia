# Text search application 
A simple text search application built with vue.js.
Userscan search through articles and matching text will be highlighted automatically 

## Features 
- Search articiles instantly 
- Filter article dynamicly 
- Clear search button 
- Responsive layout 

## Technologies uses
- HTML5
- CSS3
- JS
- Vue.js 3

index.html
style.ccss
app.js
README.md

## How to run the projecct 
Open in VS Code 
Install live server extention 
right click index.html

download the project
